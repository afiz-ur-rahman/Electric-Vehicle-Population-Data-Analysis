Electric Vehicle Population Data Analysis:

Project Overview
This project analyzes the Electric Vehicle Population dataset to understand adoption trends, vehicle types, manufacturer dominance, geographic distribution, and the role of government incentives.
The analysis is performed as part of an internship take-home assignment using Python and data analysis libraries.

Key Features:
Model Year
Make & Model
Electric Vehicle Type
Electric Range
City, County, State
CAFV Eligibility

Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

Analysis Performed:
Data cleaning and preprocessing
Trend analysis of EV adoption over years
Distribution of EV types
Top manufacturers and models
Electric range analysis
City and county-wise EV adoption
CAFV eligibility impact

Key Insights
Electric vehicle adoption has increased significantly in recent years.
Battery Electric Vehicles dominate the EV market.
A small number of manufacturers account for most EV registrations.
Urban regions show higher EV adoption.
Government incentives play a key role in encouraging EV usage.
Electric range improvements reflect technological advancement.

Conclusion
This project provides data-driven insights into electric vehicle adoption patterns and supports decision-making for policymakers, manufacturers, and infrastructure planners.
